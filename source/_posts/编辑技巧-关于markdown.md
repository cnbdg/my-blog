🌟 Markdown 进阶装修指南
1. 漂亮的折叠框（Details）
如果你的文章太长，或者想藏起一些答案/代码，可以用这个：

Markdown
<details>
<summary>点击这里查看隐藏的小秘密 (｡•̀ᴗ-)✧</summary>
哎呀，被你发现了！这里可以放很多内容哦。
</details>
效果：读者点一下文字，内容才会展开。

1. 插入带注脚的文字
想解释一个比较难的词？

这是一个比较深奥的词汇[^1]。

（在文章最下面写）：[^1]: 这里是这个词汇的详细解释哟！

🎨 针对你的 Fluid 主题的小彩蛋
因为你用的是 Fluid 主题，它其实内置了一些更漂亮的组件（这些叫 Tag Plugins），你可以直接在 .md 文件里用：

各种颜色的提示框：

Markdown
{% note success %}
这是一个成功的提示框！(๑•̀ㅂ•́)و✧
{% endnote %}

{% note danger %}
哎呀，这里出错了！(｡>ㅅ<｡)
{% endnote %}
（除了 success 和 danger，还有 info, warning, primary 等颜色哟！）

行内标签：
如果你想给某个词加个漂亮的底色，可以写：{% label primary @文字内容 %}。

🖼️ 技能二：插入你喜欢的动漫图片
在 Markdown 里插图有三种“姿势”，咱们选最简单的：

方法 A：使用图床链接（最推荐，不占空间）
你可以把动漫图片传到微博图床或 SM.MS 等网站，拿到链接后这样写：
! [图片描述](图片的网址链接)
例如：
! [我的老婆](https://example.com/anime-girl.jpg)

方法 B：存放在本地文件夹（最稳妥）
在你的 source 文件夹下创建一个叫 images 的文件夹。

把你的动漫图片放进去（假设叫 hatsune.jpg）。

在文章里写：
![初音未来](/images/hatsune.jpg)

🎨 进阶：动漫图片还能这么玩！
1. 给图片加阴影和圆角（Fluid 专属）：
你可以直接在图片链接后面加点“料”：
![描述](链接#边框-阴影-圆角)
